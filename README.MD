# React app

react app using custom webpak configuration

### How to install application

1. Install dependencies:

    ```bash
    npm install
    ```

2. How to generate bundle:

  production version:
    ```bash
    npm run package.prod
    ```

  development version with source-maps:
    ```bash
    npm run package.dev
    ```

3. How to run preview tool:

    ```bash
    npm run preview
    ```
